Detailed Description

Klasse voor het werken met de Zumo32U4 IMU-sensor.

Deze klasse bevat functies om versnelling, gyroscoopdata, kompasrichting en oriëntatie (pitch en roll) te verkrijgen, met correctie voor bias en kalibratie.
